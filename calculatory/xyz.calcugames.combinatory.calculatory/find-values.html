findValues

suspend fun findValues(grid: Array<DoubleArray>, functions: List<Operation>, sx: Int, sy: Int): Set<Double>

Find all the possible values that can be obtained by moving on a grid.

Worst-case time complexity is O((4m)^k), where m is the size of functions and k is the number of tiles that can be visited from sx and sy. This significantly decreases, as the algorithm prunes duplicate results and utilizes coroutines for parallel processing.

For example, various testing on the Mock Example, which includes a 11x11 grid with 15 functions and 28 empty tiles, took 1.5s and generated 73,118 values.

Return

A set of all the possible values that can be obtained by moving on the grid.

Parameters

grid

The grid to move on, represented as a 2D Double Array. Values of 0 represent an empty tile.

functions

The operations to apply to the values.

sx

The starting x coordinate.

sy

The starting y coordinate.
